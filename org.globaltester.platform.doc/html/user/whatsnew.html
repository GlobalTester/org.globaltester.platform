      What's new in GlobalTester 3.2.2

      • PersoSim based EAC1-ePassport simulation

        GlobalTester now includes protocol implementations to allow simulation of EAC1 ePassport based on PersoSim technology. Additionally PersoSim logging was integrated in the GlobalTester test logfiles in order to allow detailed failure analysis. Based on these changes several ActiveAuthentication related testcases have been fixed that showed minor issues with specific APDUs received from terminal.

      What's new in GlobalTester 3.2.0

      • Red is the new orange

        As the GlobalTester product and development team where acquired by secunet in 2016 several changes where made to adapt to this new situation. Most notably the design was adapted to secunet color scheme.

      • RemoteControl

        The GlobalTester provides generic remote control capabilities used for providing different services. The services which are available on your GlobalTester product are described in this category. The GlobalTester open source platform does include the generic remote control framework and GT Prove products include features to control different aspects of the tool e.g. remote testcase execution and result evaluation or simulator control.

      • Updated testspecification

        In some GT Prove products the official testspecifications where updated and the new versions are included and implemented in this release. (Note: these testcases are not included in all products)

        • RF PROTOCOL AND APPLICATION TEST STANDARD FOR EMRTD - PART 3 - TESTS FOR APPLICATION PROTOCOL AND LOGICAL DATA STRUCTURE
          version 2.10 (2016-07-07)
        • RF and Protocol Testing Part 4 - Conformity test for inspection systems
          Version 2.10, 22.06.2016

      • Several bugfixes

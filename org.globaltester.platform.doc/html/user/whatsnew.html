What's new in GlobalTester

What's new in GlobalTester 3.2.0

  • Red is the new orange

    As the GlobalTester product and development team where acquired by secunet in 2016 several changes where made to adapt to this new situation. Most notably the design was adapted to secunet color scheme.

  • RemoteControl

    The GlobalTester provides generic remote control capabilities used for providing different services. The services which are available on your GlobalTester product are described in this category. The GlobalTester open source platform does include the generic remote control framework and GT Prove products include features to control different aspects of the tool e.g. remote testcase execution and result evaluation or simulator control.

  • Updated testspecification

    In some GT Prove products the official testspecifications where updated and the new versions are included and implemented in this release. (Note: these testcases are not included in all products)

    • RF PROTOCOL AND APPLICATION TEST STANDARD FOR EMRTD - PART 3 - TESTS FOR APPLICATION PROTOCOL AND LOGICAL DATA STRUCTURE
      version 2.10 (2016-07-07)
    • Radio Frequency and Protocol Testing - Part 4 - Conformity Test for Inspection Systems
      Version 2.10, June 2016

  • Several bugfixes

What's new in GlobalTester 3.1.0

  • Integration of GT2 and GT3 for downward compatibility
  • New, intuitive user interface and handling
  • Extensive cheat sheets/user guidance
  • Service dashboard for GT processes (i.e. status of simulators)
  • Supports XML scheme according to BSI TR-03105 for test cases
  • Sample Configuration of test objects allows switching between test objects and persistence of test information
  • Easy sharing with test houses and certification authorities
  • Test Campaign allows the reproduction of a test run and persistence of test results
  • Several bugfixes

What's new in GlobalTester 3.0.2

  • User customizable exports
  • Allow upgrades of RCP product from online update site
  • Minor bugfixes

What's new in GlobalTester 3.0.1

  • Help includes now pages of SmartCardShell protocols

  • Minor bugfixes

What's new in GlobalTester 3.0.0

  • Redesigned user interface
  • Test specifications can be developed now based on new XML structure by German BSI
  • Test specification and execution in one step for faster test implementation
  • Improved monitoring of card communication through more logging and reporting features
  • Included sample test specification with samples for credit cards, ePassport, ePA, German health card, OpenPGP card
